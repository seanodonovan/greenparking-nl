## Features

### All Pages Include:
- Consistent header with topbar (contact info)
- Navigation menu (desktop and mobile)
- Mobile hamburger menu
- Footer with company info and links
- Responsive design (mobile, tablet, desktop)

### Page Descriptions:

1. **index.html** - Homepage
   - Hero section with main CTA
   - Quick booking widget
   - Features grid (4 cards)
   - Contact info in footer

2. **parkeren-bij-schiphol.html** - Parking Page
   - Hero + breadcrumb
   - Comparison with other options
   - 6 feature cards
   - Security information
   - How shuttle works (4 steps)
   - CTA banner

3. **shuttle-service.html** - Shuttle Service Page
   - 6 highlight cards
   - Route map placeholder
   - 7-step detailed process
   - Full timetable/schedule
   - Shuttle amenities grid
   - CTA banner

4. **locaties.html** - Locations Page
   - Two location cards (M & K)
   - Map placeholders
   - Address and facilities info
   - Navigation instructions
   - Location comparison table
   - Tips grid (6 items)

5. **tarieven.html** - Pricing Page
   - Shuttle parking rates table
   - Valet parking rates table
   - Seasonal pricing section (3 seasons)
   - Discounts and promotions
   - Payment methods (4 options)
   - Cancellation policy
   - FAQ section

6. **faq.html** - FAQ Page
   - 15+ detailed questions covering:
     - Shuttle frequency/operation
     - Security and safety
     - Booking and cancellation
     - Arrival procedures
     - Valet parking
     - Payment and pricing
     - Operating hours
     - Special vehicles

7. **klantenservice.html** - Customer Service Page
   - Contact information card
   - Contact form (name, email, phone, subject, message)
   - Contact methods details
   - Service areas grid (6 items)
   - Response times section
   - Language information

8. **reserveren.html** - Reservation/Booking Page
   - Comprehensive booking form with:
     - Parking type selection
     - Date/time inputs
     - License plate
     - Personal information
     - Optional flight number
     - Automatic price calculation
   - Price display
   - 6-step process explanation
   - Info sidebar with tips
   - Form validation with JavaScript
